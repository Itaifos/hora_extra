# Sistema de Gestão de Horas Extras e Benefícios

Sistema para registrar, processar e converter horas extras dos funcionários em saldo de benefício interno para uso no restaurante da empresa.

## Estrutura do Projeto

- **backend/**: API em Node.js (NestJS/Express) e regras de negócio.
- **frontend/**: Interface Web em React.
- **documentacao-app-hora-extra.pdf**: Documentação de requisitos e regras de negócio.
- **PLANEJAMENTO.md**: Roadmap de desenvolvimento.

## Como Iniciar

Instruções de instalação e execução serão adicionadas conforme o desenvolvimento avança.
